  1. object DataOnWorker extends Serializable

    It is a singleton.

    It is a singleton. It maintains in each machine the Matrix to collect the information to compute the best cuts. In each iteration:

    • the first thread accessing the singleton initializes the Matrix
    • all the partitions concurrently update the matrix during the "Local Information Collection" step
    • during the "Distributed Information Aggregation" step the fist thread entering is the one in charge of sending the data to the shuffling phase
    • the Matrix is freed
